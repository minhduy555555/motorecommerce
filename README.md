## Getting Started
```bash
The Technologies used:
    NextJs
    Tailwindcss
    React icons
```

First, install and setting framework:

```bash
# React icons
npm i react-icons
# Setting for .eslintrc.json if get error cannot find module "next/babel"
{
  "extends": ["next/babel","next/core-web-vitals"]
}
```

Second, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```# motorecommerce
